2.3 Density Phase Transitions

Definition 2.3.1 Density

The density of a Subset Sum instance \((a_1, \ldots , a_n)\) is \(d = n / \log _2(\max _i a_i)\).

Theorem 2.3.2 Density Dichotomy

def:density

  • Low density (\(d {\lt} 0.94\)): solvable in polynomial time via lattice reduction (Lagarias–Odlyzko, CJLOSS).

  • High density (\(d \gg 1\)): solvable by pigeonhole—\(2^n\) subsets exceed the number of possible sums.

  • Critical density (\(d \approx 1\)): believed to be maximally hard.
