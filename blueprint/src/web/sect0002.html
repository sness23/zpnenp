1.1 Subset Sum

Definition 1.1.1 Subset Sum

SubsetSum Let \(S\) be a finite set of integers and \(t \in \ZZ \). The Subset Sum predicate \(\mathrm{SubsetSum}(S, t)\) holds if there exists \(S' \subseteq S\) such that \(\sum _{x \in S'} x = t\).

Definition 1.1.2 Zero-Sum Subset

SubsetSumZero def:subset_sum \(\mathrm{SubsetSumZero}(S)\) holds if there exists a nonempty \(S' \subseteq S\) with \(\sum _{x \in S'} x = 0\).

Definition 1.1.3 Achievable Subset Sums

subsetSums def:subset_sum The set of all achievable subset sums of \(S\):

\[ \mathrm{subsetSums}(S) = \left\{ \sum _{x \in S'} x \; \middle |\; S' \subseteq S \right\} . \]
Theorem 1.1.4 Subset Sum at Zero

subsetSum_zero def:subset_sum For any finite set \(S\) of integers, \(\mathrm{SubsetSum}(S, 0)\) holds (witnessed by the empty subset).

Theorem 1.1.5 Monotonicity

subsetSum_mono def:subset_sum If \(S \subseteq T\) and \(\mathrm{SubsetSum}(S, t)\), then \(\mathrm{SubsetSum}(T, t)\).

Theorem 1.1.6 Singleton

subsetSum_singleton def:subset_sum \(\mathrm{SubsetSum}(\{ a\} , t)\) if and only if \(t = 0\) or \(t = a\).

Theorem 1.1.7 Counting Bound

card_subsetSums_le def:subset_sums \(|\mathrm{subsetSums}(S)| \leq 2^{|S|}\).
