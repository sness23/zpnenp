1.2 Modular Subset Sum and EGZ

Definition 1.2.1 Modular Zero-Sum

ModSubsetSumZero \(\mathrm{ModSubsetSumZero}(S, n)\) holds if there exists a nonempty \(S' \subseteq S\) with \(n \mid \sum _{x \in S'} x\).

Theorem 1.2.2 EGZ for Finsets

egz_finset def:mod_subset_sum_zero If \(|S| \geq 2n - 1\) and \(n {\gt} 0\), then there exists \(T \subseteq S\) with \(|T| = n\) and \(n \mid \sum _{x \in T} x\).

Theorem 1.2.3 EGZ Implies Modular Zero-Sum

egz_implies_modSubsetSumZero thm:egz_finset, def:mod_subset_sum_zero If \(|S| \geq 2n - 1\) and \(n {\gt} 1\), then \(\mathrm{ModSubsetSumZero}(S, n)\).

Significance: The adversary cannot construct a set of \(\geq 2n-1\) integers that avoids having an \(n\)-element subset with sum divisible by \(n\). Existence is forced by combinatorial structure.
